# Tap

Tap for my personal macOS applications:

- [Reedsy for Mac](https://github.com/aubreypwd/reedsy-mac/), `reedsy-mac`

## How do I install these formulae?

```bash
brew tap aubreypwd/homebrew-cask
brew update
brew cask install <formula>
```
